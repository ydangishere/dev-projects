## AI Backend Job Finder

Desktop app that scrapes job listings from TopCV or ITviec, uses an LLM to identify backend-related jobs, then shows results in a window.

- **Tech**: Python, Playwright (for ITviec), OpenAI API, Tkinter UI.
- **Original repo**: `https://github.com/ydangishere/aibackendjob_finder`

For installation and full usage steps (Windows, Mac, Docker/FlareSolverr for ITviec), see the original README on GitHub.
