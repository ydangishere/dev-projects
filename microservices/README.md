## Microservices

Fundamental yet complete microservices system with Auth, People, and Case services, plus Admin UI.

- **Tech**: Node.js, TypeScript, Express, PostgreSQL, Redis, Kafka, Elasticsearch, Docker.
- **Services**:
  - Auth Service (JWT auth, register, login, profile)
  - People Service (CRUD, Redis cache, Kafka events)
  - Case Service (CRUD, Elasticsearch search, Kafka consumer)
- **Original repo**: `https://github.com/ydangishere/microservices`

See the original README for full architecture diagrams, local/Render setup steps, and API examples.
