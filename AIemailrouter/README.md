## AI Email Router

Desktop app that reads plain-text emails, uses an LLM to classify intent (sales, support, billing, spam, other), and can send results to Telegram.

- **Tech**: Python, Tkinter UI, OpenAI API, optional Telegram bot integration.
- **Usage**: Paste OpenAI API key, paste emails (one per line), click Analyze, optionally configure Telegram bot token and recipient to send results.
- **Original repo**: `https://github.com/ydangishere/AIemailrouter`

For detailed step-by-step usage and Telegram setup, see the original README on GitHub.
