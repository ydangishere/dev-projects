# Gig 1 – Shopify Speed & Conversion — Lưu ý

## Dùng theme Dawn (free)

- **Được và nên dùng** theme Dawn trong store của bạn.
- Dawn = theme free chính thức của Shopify. Vào **Online Store → Themes → Add theme → Dawn**. Không cần tải từ nguồn khác.
- **Lý do:** Chuẩn Shopify (index tốt), code sạch dễ tối ưu, client hay dùng Dawn hoặc theme tương tự. **Không cần theme trả phí** — freelancer mới dùng Dawn là đủ.

---

## Case “Speed Optimization” phải có before/after rõ

- **Đừng** tối ưu Dawn bản sạch — Dawn mặc định đã khá nhanh → số liệu kiểu 85 → 90 không đủ ấn tượng.
- **Nên:** Cố ý làm site chậm một chút rồi mới tối ưu:
  - Cài **1–2 app nặng** (popup, review, tracking…)
  - Thêm **hero image lớn**
  - Thêm **vài script test**
  - Đo (vd: 55–60) → tối ưu → đo lại (vd: 85–90).
- Mục tiêu: **55 → 85** hoặc **60 → 90** để case study thuyết phục.
