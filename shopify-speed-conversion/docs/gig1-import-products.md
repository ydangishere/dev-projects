# Gig 1 — Thêm 5–10 sản phẩm demo (store nặng)

## File: `gig1-products-import.csv`

- Có sẵn **10 sản phẩm** (Demo Product 1 → 10), giá mẫu.
- Cột **Image Src** để trống — bạn thêm ảnh sau.

## Cách làm

### 1. Import CSV vào Shopify

1. Vào **Shopify Admin** → **Products** → **Import**.
2. Chọn file `gig1-products-import.csv` (nằm trong folder `docs` của project).
3. Import xong sẽ có 10 products (chưa có ảnh).

### 2. Thêm ảnh lớn (chưa tối ưu) cho từng sản phẩm

- Vào từng product → **Media** → **Add image**.
- Dùng ảnh **đủ lớn** (vd: 1500×1500 trở lên, file 500KB–2MB) — chưa cần compress.
- Có thể tải ảnh từ Unsplash/Pexels (bản gốc, không resize nhỏ) rồi upload.

Hoặc dùng **Bulk editor:** Products → chọn nhiều product → Edit products → thêm ảnh cho từng cái.

### 3. Kết quả

- Store có 5–10 sản phẩm với ảnh nặng → trang collection/product sẽ chậm hơn → đo **before** (PageSpeed/GTmetrix) rồi mới tối ưu.
