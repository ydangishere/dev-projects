## Dev Projects

This monorepo collects several backend and AI-related projects from the `ydangishere` GitHub account.

### Included sub-projects

- **AI Backend Job Finder** (`aibackendjob_finder`): Desktop app that scrapes TopCV/ITviec job listings and uses an LLM to filter backend positions. Original repo: `https://github.com/ydangishere/aibackendjob_finder`.
- **Payment Transaction API** (`payment-transaction-api`): FastAPI-based payment/order API with idempotency keys, PostgreSQL, and Redis caching. Original repo: `https://github.com/ydangishere/payment-transaction-api`.
- **Microservices** (`microservices`): Node.js/TypeScript microservices system (Auth, People, Case) with PostgreSQL, Redis, Kafka, and Elasticsearch. Original repo: `https://github.com/ydangishere/microservices`.
- **AI Email Router** (`AIemailrouter`): Python desktop app that classifies emails by intent using an LLM and can send results to Telegram. Original repo: `https://github.com/ydangishere/AIemailrouter`.
- **Case Mini Backend** (`case-mini-backend`): Java/Spring Boot mini case/order backend with CRUD, validation, pagination, and Redis caching. Original repo: `https://github.com/ydangishere/case-mini-backend`.

Each subfolder in this repo corresponds to one of the original projects and contains its own `README.md` with a short description and a link back to the source repository.
